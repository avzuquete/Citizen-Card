# Applications developed for the Portuguese eID (Citizen Card, or Cartão de Cidadão)

1. Authentication with the CC (**ccauth**)
2. Application that dumps many of the information of a CC (both through PTEID and PKCS #11 libraries, **ccdump**)
3. PAM module to authenticate users with their CC (**ccpam**)
4. Validation of the certificates carried by a Citizen Card (**ccval**). 
5. Wrapper for the CC PKCS #11 library (**pkcs-11-wrapper**).

# Applications developed for the Portuguese eID (Citizen Card, or Cartão de Cidadão)

1. Authentication with the CC (**ccauth**)
2. PAM module to authenticate users with their CC (**ccpam**)
3. Validation of the certificates carried by a Citizen Card (**ccval**). 
4. Wrapper for the CC PKCS #11 library (**pkcs-11-wrapper**).
